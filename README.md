# 🧠 Memnexa

> Your personal AI knowledge operating system.

Memnexa is a modern AI-powered knowledge management platform designed to help users organize notes, documents, websites, and ideas into a searchable knowledge base. It combines a beautiful UI with AI-assisted workflows to make knowledge retrieval fast and intuitive.

> 🚧 **Project Status:** Active Development

---

## ✨ Features

### ✅ Currently Working

- Responsive modern UI
- Landing page
- Dark / Light theme
- Animated interface
- Authentication UI
- Knowledge workspace UI
- Dashboard layout
- AI chat interface (UI)
- Smooth page transitions
- Responsive design
